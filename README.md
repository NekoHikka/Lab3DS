# Blockchain — платіжна система

Мінімальна реалізація блокчейну на JavaScript (один HTML-файл, без залежностей).

## Огляд

Система для обліку платежів між користувачами. Акаунти можуть отримувати монети, підписувати та надсилати платежі. Транзакції групуються в блоки, кожен з яких криптографічно прив'язаний до попереднього через хеш.

## Функції

1. Створення акаунтів з ключовою парою
2. Поповнення балансу через faucet
3. Відправка підписаного платежу
4. Формування блоку з транзакцій та валідація перед додаванням до ланцюжка

## Безпека

Цілісність ланцюжка забезпечується через prevHash. Кожна операція підписана приватним ключем відправника. Є захист від подвійного витрачання та від дублювання транзакцій через nonce і txDatabase. Підпис побудований на SHA-1 — для навчальних цілей.

## Обмеження

Дані зберігаються лише в пам'яті браузера. Немає децентралізації, консенсусу та збереження між сесіями.

## Запуск

Відкрити `index.html` у браузері. Сервер не потрібен.
